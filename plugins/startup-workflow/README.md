# Startup Workflow Plugin

This plugin packages the staged 0-1 product initiation workflow from this repository as a Codex plugin.

It includes:

- the `a1-startup-workflow-orchestrator` control-plane skill
- seven stage skills from opportunity framing to MVP scope
- plugin metadata for installation and discovery
- local marketplace registration for repo-local use

Use this plugin when you want the workflow to behave like an installable Codex extension rather than a loose set of skills.
